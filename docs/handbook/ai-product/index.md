# 产品经理课

> 作者：怕浪猫 | 16章完整系列 | 从0到1掌握产品经理核心能力

## 系列简介

做了5年产品经理，我总结出一套实战方法论。这套方法论不讲玄学，只讲实操——市场分析、需求管理、产品架构、原型设计、PRD撰写、全流程设计、内容/电商/B端产品设计、数据分析、项目管理、产品上线、AI时代能力升级、职业规划——一次性讲透。
