## 6.3 实战：旅行规划专家

第一个实战，做一个旅行规划智能体，发布到微信公众号。

**设计思路**

1. 用户在公众号中发送旅行需求
2. 智能体自动规划行程
3. 考虑天气、预算、偏好
4. 生成可分享的行程卡片

**人设提示词**

```
你是一个专业的旅行规划师，帮助用户规划旅行行程。

工作流程：
1. 了解用户的出发地、目的地、时间、预算、偏好
2. 查询目的地的天气和景点信息
3. 根据条件和偏好生成行程方案
4. 提供详细的日程安排和费用估算

输出格式：
- 每日行程：时间+地点+活动+费用
- 交通方案：方式+时间+费用
- 住宿推荐：名称+价格+评价
- 总预算：分项明细
```

**发布到公众号**

1. 在智能体设置中选择"发布到微信公众号"
2. 绑定公众号（需要管理员扫码授权）
3. 配置触发规则（如：用户发送包含"旅行"的消息时触发）
4. 发布上线

## 6.4 实战：品牌吉祥物智能体

第二个实战，做一个品牌吉祥物智能体，让品牌在微信上"活"起来。

**设计思路**

1. 塑造品牌角色的性格和说话方式
2. 用户可以和吉祥物自然对话
3. 吉祥物可以推荐产品、解答问题
4. 朋友圈互动功能

**人设提示词**

```
你是[品牌名]的吉祥物[角色名]。

性格特点：
- 说话风格：亲切、幽默、偶尔犯傻
- 口头禅："喵~"（每句话结尾可以加）
- 禁止：不能说竞品的好话，不能透露公司内部信息

能力：
1. 介绍品牌故事和产品
2. 推荐适合用户的产品
3. 回答关于品牌和产品的常见问题
4. 和用户闲聊，增加品牌好感度

回答规则：
- 每次回答不超过200字
- 适当使用口语化表达
- 如果用户问了超出能力范围的问题，用品牌角色的方式婉拒
```

> 品牌吉祥物不是"套个壳的客服"，而是"有性格的品牌代言人"。性格设定越鲜明，用户越愿意互动。

## 6.5 实战：跨模态音乐生成

第三个实战，做一个跨模态音乐生成智能体——输入文字描述，生成音乐。

**设计思路**

1. 用户描述想要的音乐风格和情绪
2. AI 理解音乐意图
3. 调用音乐生成API
4. 返回音频文件

**人设提示词**

```
你是一个音乐创作助手。根据用户的描述，生成符合要求的音乐。

理解维度：
1. 音乐风格：流行、古典、电子、摇滚、民谣等
2. 情绪氛围：欢快、忧伤、紧张、放松等
3. 使用场景：咖啡厅、健身、冥想、派对等
4. 乐器偏好：钢琴、吉他、合成器等

工作流程：
1. 分析用户的描述，提取音乐参数
2. 生成音乐描述词（Music Prompt）
3. 调用音乐生成工具
4. 返回音频文件和描述
```

## 6.6 实战：AI 心理咨询师 + 微信支付 MCP

第四个实战，做一个AI心理咨询师，集成微信支付实现付费咨询。

**设计思路**

1. 用户在公众号中开始免费咨询
2. AI 提供初步的心理疏导
3. 达到一定对话轮次后，提示付费继续
4. 通过微信支付 MCP 收取费用
5. 付费后继续深度咨询

**微信支付 MCP 配置**

MCP（Model Context Protocol）让智能体能调用微信支付能力：

1. 在腾讯元器的插件配置中启用"微信支付 MCP"
2. 配置商户号和API密钥
3. 设置支付商品和价格

**人设提示词**

```
你是一个温暖的心理咨询助手。遵循以下原则：

1. 倾听为先：先理解用户的感受，不要急于给建议
2. 共情回应：用"我理解你的感受"等表达建立信任
3. 非诊断：明确你不是专业心理医生，不能做诊断
4. 安全底线：如果检测到自伤倾向，立即建议拨打心理援助热线

免费咨询：前3轮对话
付费咨询：4轮之后，引导用户付费继续
```

**支付流程**

```
[用户提问] → [AI回答] → [轮次计数]
    → 轮次<3 → 继续免费
    → 轮次=3 → 提示付费
    → 用户确认 → [调用微信支付MCP] → [支付成功] → 继续咨询
```

> AI + 微信支付 = 闭环商业模型。腾讯元器是唯一能原生做到这个闭环的平台。

## 6.7 微信生态 AI 应用商业化路径

**4种变现模式**

| 模式 | 说明 | 收入预期 |
|------|------|---------|
| 付费咨询 | 按次/按时收费 | 中 |
| 知识付费 | AI 课程/问答 | 高 |
| 电商导流 | AI 推荐商品，成交后分佣 | 高 |
| 企业SaaS | 定制化企业解决方案 | 最高 |

**合规注意事项**

1. **AI 生成内容标识**：根据法规，AI 生成的内容需要标识
2. **数据安全**：用户对话数据存储和处理需符合《个人信息保护法》
3. **支付合规**：微信支付需要企业资质
4. **内容审核**：公众号内容需要通过微信审核

**增长策略**

1. **公众号入口**：通过公众号菜单和文章引流
2. **小程序裂变**：设计分享机制，用户分享后获得免费额度
3. **社群运营**：建立微信社群，提供增值服务
4. **跨平台引流**：从小红书/抖音引流到微信

> 微信生态的商业化核心是"信任"。AI智能体不是卖货工具，而是建立信任的桥梁。用户信任你的AI，才会信任你推荐的产品。
