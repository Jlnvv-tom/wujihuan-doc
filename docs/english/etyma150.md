# 词根 100-150

## 🔹 101. cred – 相信

来源：拉丁语 credere，“相信”。

历史典故：credible 是“值得相信的”，incredible 是“难以置信的”。

常见单词：credible（可信的）、incredible（难以置信的）、credit（信用）、credo（信条）

记忆提示：cred = 相信；credible 是“可以相信的”。
