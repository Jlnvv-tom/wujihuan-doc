# 词根 1-50

## 🔹51. port – 携带，运送

来源：拉丁语 portare，意思是“携带、运输”。

历史典故：罗马士兵携带物资行军，用“portable”容器装载，“transport”是作战的关键。

常见单词：export（出口）、import（进口）、portable（便携的）、transport（运输）

记忆提示：port = 搬运；portable 就是可以带走的。

## 🔹52. form – 形状，形成

来源：拉丁语 forma，指“形状、结构”。

历史典故：古罗马哲学认为“form”是事物本质的体现，影响后来的形而上学理论。

常见单词：uniform（统一的）、transform（变形）、inform（通知/使成形）、reform（改革）

记忆提示：form 是“形”，transform 是变形金刚的“变形”。

## 🔹53. cap / capt / cept / cip – 拿，抓

来源：拉丁语 capere，意思是“抓住、获取”。

历史典故：capture（捕捉）最早用于战斗或狩猎，concept 是“抓住思想的产物”。

常见单词：capture（捕获）、accept（接受）、intercept（拦截）、recipient（接收者）

记忆提示：cap = 抓；accept = 抓住机会 = 接受。

## 🔹54. path – 感情，疾病

来源：希腊语 pathos，意思是“感情、痛苦、疾病”。

历史典故：pathos 是古希腊修辞术之一，强调通过情感打动人。

常见单词：empathy（共情）、sympathy（同情）、pathology（病理学）、antipathy（反感）

记忆提示：path 有“情绪或痛”的意味；sympathy 是“共同感受”。

## 🔹55. phon – 声音

来源：希腊语 phōnē，意为“声音”。

历史典故：最早与语言学和诗歌结构有关，强调“声音之美”。

常见单词：telephone（电话）、microphone（麦克风）、phonics（拼音）、symphony（交响乐）

记忆提示：phon = 声音；telephone 是“远方的声音”。

## 🔹56. scrib / script – 写

来源：拉丁语 scribere，意思是“写”。

历史典故：scriptus 是“已写下的”，是早期文献记录的重要形式。

常见单词：describe（描述）、manuscript（手稿）、prescription（处方）、subscribe（订阅）

记忆提示：script 就是“写的东西”；subscribe = 写下名字订阅。

## 🔹57. sect / sec / seg – 切，割

来源：拉丁语 secare，意思是“切”。

历史典故：sect 最早是“分离的小群体”，如宗派；segment 是被分出来的一块。

常见单词：section（部分）、intersect（相交）、dissect（解剖）、segment（段）

记忆提示：sect = 切，dissect 是切开进行研究。

## 🔹58. therm – 热

来源：希腊语 thermē，意思是“热”。

历史典故：thermal 原用于描述罗马温泉，thermometer 是热学革命的成果之一。

常见单词：thermal（热的）、thermometer（温度计）、thermodynamics（热力学）

记忆提示：therm = 热，thermometer = 热的测量器。

## 🔹59. vert / vers – 转

来源：拉丁语 vertere，意思是“转动”。

历史典故：convert（转变）最早指宗教改信，divert（转移）是策略变化。

常见单词：convert（转换）、divert（转移）、reverse（反转）、advertise（宣传）

记忆提示：vers = 转，advertise 是“把注意力转过来”。

## 🔹60. mort – 死亡

来源：拉丁语 mors / mortis，意为“死亡”。

历史典故：mortuary（停尸房）和 mortal（凡人）都反映了古人对死亡的敬畏。

常见单词：mortal（致命的）、mortality（死亡率）、immortal（不朽的）、mortify（羞辱到极点）

记忆提示：mort = 死，immortal = 不死的。

## 🔹61. duc / duct – 引导，带领

来源：拉丁语 ducere，意思是“引导、领导”。

历史典故：在罗马，duke（公爵）是“引导士兵的人”，conduct 是“引领行为”。

常见单词：conduct（引导/行为）、educate（教育）、introduce（介绍）、reduce（减少）

记忆提示：duc = 带，educate 是“向外引导”出潜能。

## 🔹62. pel / puls – 推，驱动

来源：拉丁语 pellere，意为“驱动、推动”。

历史典故：在军事和动力领域都很常见，如 compel（强迫）或 repel（击退敌人）。

常见单词：compel（强迫）、propel（推进）、repulse（击退）、expel（驱逐）

记忆提示：pel = 推；propel = 往前推。

## 🔹63. man / manu – 手

来源：拉丁语 manus，意思是“手”。

历史典故：manipulate 来自“用手操作”，manuscript（手稿）指亲手写下的东西。

常见单词：manual（手工的）、manipulate（操纵）、manufacture（制造）、manuscript（手稿）

记忆提示：manu = 手；manual 是“用手做的”。

## 🔹64. fid – 信任

来源：拉丁语 fides，意思是“信任、忠诚”。

历史典故：在古罗马契约与誓言中，“fides”是极重要的价值。

常见单词：confide（吐露）、fidelity（忠诚）、infidel（不信教者）、confident（自信的）

记忆提示：fid = 信；fidelity 是“对信任的忠实”。

## 🔹65. fin – 结束，边界

来源：拉丁语 finis，意为“终点、界限”。

历史典故：fin 与“完成”有关，如 final（最终）、finish（结束）都含“终点”之意。

常见单词：final（最终的）、define（定义）、infinite（无限的）、refine（精炼）

记忆提示：fin = 终点；define 是“划出边界”。

## 🔹66. rupt – 破裂

常见单词：rupture（破裂）、interrupt（打断）、erupt（爆发）、corrupt（腐败的）

记忆提示：rupt = 破；erupt 是“爆出来”。

## 🔹67. ven / vent – 来

记忆提示：ven = 来；intervene = 插进来。

## 🔹71. tend / tens / tent – 拉伸，延展

## 🔹72. struct – 建造，堆叠

## 🔹73. cred – 相信

常见单词：credit（信用）、credible（可信的）、incredible（难以置信的）、discredit（怀疑）

记忆提示：cred = 信；credit 是别人“相信你”的表示。

## 🔹 100. clud / clus – 关闭

来源：拉丁语 claudere，意思是“关闭”。

历史典故：exclude 是“排除”，reclude 是“关闭起来”。

常见单词：exclude（排除）、conclude（结束）、enclose（围住）、include（包括）

记忆提示：clud = 关闭；exclude 是“关闭在外面”。
