词根 1-50 ​

🔹51. port – 携带，运送 ​

来源：拉丁语 portare，意思是“携带、运输”。

历史典故：罗马士兵携带物资行军，用“portable”容器装载，“transport”是作战的关键。

常见单词：export（出口）、import（进口）、portable（便携的）、transport（运输）

记忆提示：port = 搬运；portable 就是可以带走的。

🔹52. form – 形状，形成 ​

来源：拉丁语 forma，指“形状、结构”。

历史典故：古罗马哲学认为“form”是事物本质的体现，影响后来的形而上学理论。

常见单词：uniform（统一的）、transform（变形）、inform（通知/使成形）、reform（改革）

记忆提示：form 是“形”，transform 是变形金刚的“变形”。

🔹53. cap / capt / cept / cip – 拿，抓 ​

来源：拉丁语 capere，意思是“抓住、获取”。

历史典故：capture（捕捉）最早用于战斗或狩猎，concept 是“抓住思想的产物”。

常见单词：capture（捕获）、accept（接受）、intercept（拦截）、recipient（接收者）

记忆提示：cap = 抓；accept = 抓住机会 = 接受。

🔹54. path – 感情，疾病 ​

来源：希腊语 pathos，意思是“感情、痛苦、疾病”。

历史典故：pathos 是古希腊修辞术之一，强调通过情感打动人。

常见单词：empathy（共情）、sympathy（同情）、pathology（病理学）、antipathy（反感）

记忆提示：path 有“情绪或痛”的意味；sympathy 是“共同感受”。

🔹55. phon – 声音 ​

来源：希腊语 phōnē，意为“声音”。

历史典故：最早与语言学和诗歌结构有关，强调“声音之美”。

常见单词：telephone（电话）、microphone（麦克风）、phonics（拼音）、symphony（交响乐）

记忆提示：phon = 声音；telephone 是“远方的声音”。

🔹56. scrib / script – 写 ​

来源：拉丁语 scribere，意思是“写”。

历史典故：scriptus 是“已写下的”，是早期文献记录的重要形式。

常见单词：describe（描述）、manuscript（手稿）、prescription（处方）、subscribe（订阅）

记忆提示：script 就是“写的东西”；subscribe = 写下名字订阅。

🔹57. sect / sec / seg – 切，割 ​

来源：拉丁语 secare，意思是“切”。

历史典故：sect 最早是“分离的小群体”，如宗派；segment 是被分出来的一块。

常见单词：section（部分）、intersect（相交）、dissect（解剖）、segment（段）

记忆提示：sect = 切，dissect 是切开进行研究。

🔹58. therm – 热 ​

来源：希腊语 thermē，意思是“热”。

历史典故：thermal 原用于描述罗马温泉，thermometer 是热学革命的成果之一。

常见单词：thermal（热的）、thermometer（温度计）、thermodynamics（热力学）

记忆提示：therm = 热，thermometer = 热的测量器。

🔹59. vert / vers – 转 ​

来源：拉丁语 vertere，意思是“转动”。

历史典故：convert（转变）最早指宗教改信，divert（转移）是策略变化。

常见单词：convert（转换）、divert（转移）、reverse（反转）、advertise（宣传）

记忆提示：vers = 转，advertise 是“把注意力转过来”。

🔹60. mort – 死亡 ​

来源：拉丁语 mors / mortis，意为“死亡”。

历史典故：mortuary（停尸房）和 mortal（凡人）都反映了古人对死亡的敬畏。

常见单词：mortal（致命的）、mortality（死亡率）、immortal（不朽的）、mortify（羞辱到极点）

记忆提示：mort = 死，immortal = 不死的。

🔹61. duc / duct – 引导，带领 ​

来源：拉丁语 ducere，意思是“引导、领导”。

历史典故：在罗马，duke（公爵）是“引导士兵的人”，conduct 是“引领行为”。

常见单词：conduct（引导/行为）、educate（教育）、introduce（介绍）、reduce（减少）

记忆提示：duc = 带，educate 是“向外引导”出潜能。

🔹62. pel / puls – 推，驱动 ​

来源：拉丁语 pellere，意为“驱动、推动”。

历史典故：在军事和动力领域都很常见，如 compel（强迫）或 repel（击退敌人）。

常见单词：compel（强迫）、propel（推进）、repulse（击退）、expel（驱逐）

记忆提示：pel = 推；propel = 往前推。

🔹63. man / manu – 手 ​

来源：拉丁语 manus，意思是“手”。

历史典故：manipulate 来自“用手操作”，manuscript（手稿）指亲手写下的东西。

常见单词：manual（手工的）、manipulate（操纵）、manufacture（制造）、manuscript（手稿）

记忆提示：manu = 手；manual 是“用手做的”。

🔹64. fid – 信任 ​

来源：拉丁语 fides，意思是“信任、忠诚”。

历史典故：在古罗马契约与誓言中，“fides”是极重要的价值。

常见单词：confide（吐露）、fidelity（忠诚）、infidel（不信教者）、confident（自信的）

记忆提示：fid = 信；fidelity 是“对信任的忠实”。

🔹65. fin – 结束，边界 ​

来源：拉丁语 finis，意为“终点、界限”。

历史典故：fin 与“完成”有关，如 final（最终）、finish（结束）都含“终点”之意。

常见单词：final（最终的）、define（定义）、infinite（无限的）、refine（精炼）

记忆提示：fin = 终点；define 是“划出边界”。

🔹66. rupt – 破裂 ​

来源：拉丁语 rumpere，意思是“破裂”。

历史典故：rupture 原为“血管或关系的破裂”，corrupt（腐败）意指“内部破裂”。

常见单词：rupture（破裂）、interrupt（打断）、erupt（爆发）、corrupt（腐败的）

记忆提示：rupt = 破；erupt 是“爆出来”。

🔹67. ven / vent – 来 ​

来源：拉丁语 venire，意思是“来”。

历史典故：adventure（冒险）本意是“来临的事情”，event 是“发生的事”。

常见单词：adventure（冒险）、convention（大会）、intervene（干预）、event（事件）

记忆提示：ven = 来；intervene = 插进来。

🔹68. vers / vert – 转（再次出现，常见词根） ​

来源：拉丁语 vertere，与先前第 59 条重复，但值得再次强调。

常见单词：version（版本）、introvert（内向者）、extrovert（外向者）、invert（倒转）

记忆提示：vers = 转，introvert 是“向内转”。

🔹69. voc / vok – 呼唤，声音 ​

来源：拉丁语 vocare，意思是“呼唤”。

历史典故：vocation 是“被呼召去做的事”，invoke 是“呼求（神）”。

常见单词：vocal（发声的）、advocate（提倡）、vocabulary（词汇量）、evoke（唤起）

记忆提示：voc = 呼；vocal 是“声音的”。

🔹70. sign – 记号，标志 ​

来源：拉丁语 signum，意思是“标记、信号”。

历史典故：在罗马军团中，“signum”是旗帜、象征身份。

常见单词：signal（信号）、signature（签名）、design（设计）、assign（分配）

记忆提示：sign = 标志；signature 是“你的标志”。

🔹71. tend / tens / tent – 拉伸，延展 ​

来源：拉丁语 tendere，意思是“拉紧、延展”。古罗马士兵拉弓时常用此词。

历史典故：tension（紧张）原意是“弦拉紧状态”，tendency 是“向某方向延展的趋势”。

常见单词：extend（扩展）、tension（紧张）、attend（出席）、tendency（倾向）

记忆提示：tend = 拉；tendency 是“拉向某种倾向”。

🔹72. struct – 建造，堆叠 ​

来源：拉丁语 struere，意思是“堆积、建构”，与罗马工程和军事营建相关。

历史典故：construct（构建）和 instruct（传授）都含有“有系统地搭建”的含义。

常见单词：structure（结构）、construct（建造）、instruct（指导）、destruction（破坏）

记忆提示：struct = 构造；destruction 是“结构的瓦解”。

🔹73. cred – 相信 ​

来源：拉丁语 credere，意思是“相信、信任”。在古代债务交易中至关重要。

历史典故：credit 最初意为“别人相信你会还钱”，credible 是“能被相信的”。

常见单词：credit（信用）、credible（可信的）、incredible（难以置信的）、discredit（怀疑）

记忆提示：cred = 信；credit 是别人“相信你”的表示。

🔹74. ject – 投，掷 ​

来源：拉丁语 jacere，意思是“投掷”。中世纪战争术语中常用于抛射器。

历史典故：inject 是“投进去（药）”，project 是“向前投（计划/影像）”。

常见单词：inject（注射）、reject（拒绝）、project（项目/投影）、eject（驱逐）

记忆提示：ject = 扔；eject 是“扔出去”。

🔹75. log / logue – 言语，话语 ​

来源：希腊语 logos，意思是“言辞、道理”。哲学中表示“理性语言”。

历史典故：dialogue 是“两人对话”，logic 是“有道理的言说方式”。

常见单词：dialogue（对话）、monologue（独白）、logic（逻辑）、apology（道歉）

记忆提示：log = 言；dialogue 是“两人的语言”。

🔹76. meter / metr – 测量 ​

来源：希腊语 metron，意思是“度量衡、标准”。用于科学计量单位。

历史典故：thermometer 是“测热器”，barometer 是“测气压器”。

常见单词：thermometer（温度计）、diameter（直径）、speedometer（速度计）、barometer（气压计）

记忆提示：meter = 测；thermometer 是“测温工具”。

🔹77. ast / astr – 星星 ​

来源：希腊语 astron，意思是“星辰”。常用于占星和天文学。

历史典故：astronomy 是“研究星体”，disaster 源自“不吉的星象（dis + aster）”。

常见单词：astronomy（天文学）、astrology（占星学）、astronaut（宇航员）、disaster（灾难）

记忆提示：astr = 星；astronaut 是“航行星际者”。

🔹78. phon – 声音 ​

来源：希腊语 phone，意为“声音”。古希腊修辞术强调“音韵之美”。

历史典故：symphony 是“共同发出的声音”，microphone 是“扩音器”。

常见单词：telephone（电话）、microphone（麦克风）、symphony（交响乐）、phonetic（语音的）

记忆提示：phon = 声；telephone 是“远距离的声音”。

🔹79. port – 携带，运输 ​

来源：拉丁语 portare，意思是“携带”。古代用于指士兵运送军备。

历史典故：export 是“运出”，import 是“运入”，transport 是“横向运送”。

常见单词：export（出口）、import（进口）、transport（运输）、portable（可携带的）

记忆提示：port = 携带；portable 是“可带走的”。

🔹80. scrib / script – 写 ​

来源：拉丁语 scribere，意为“写作”。古代文书书记称为“scribe”。

历史典故：manuscript 是“手写稿”，describe 是“下笔写出”。

常见单词：describe（描述）、manuscript（手稿）、script（剧本）、subscribe（订阅）

记忆提示：scrib = 写；manuscript 是“用手写的”。

🔹 81. vid / vis – 看见 ​

来源：拉丁语 videre，意思是“看到”。video 最初是“我看见”的意思。

历史典故：visible 和 revise 都与“看”相关，revise 原意为“再看一遍，检查”。

常见单词：video（视频）、visible（可见的）、revise（修改）、vision（视野）

记忆提示：vid = 视；video 是“影像，看到的内容”。

🔹 82. tele – 远距离 ​

来源：希腊语 tēle，意为“远”。在古希腊文化中常用于描述远距离的通信幻想。

历史典故：television 意指“远距离的视觉”，而电话（telephone）则是远距离的声音传递。

常见单词：telephone（电话）、television（电视）、teleport（瞬移）、telegraph（电报）

记忆提示：tele = 远；television 是“远距离的影像”。

🔹 83. form – 形状，形成 ​

来源：拉丁语 forma，意为“形状、外貌”。

历史典故：reform 原指“对事物形状的改变”，transform 是“彻底的形状变化”。

常见单词：reform（改革）、transform（变形）、inform（通知）、uniform（制服/一致的）

记忆提示：form = 形状；uniform 是“一致的形状”。

🔹 84. rupt – 破裂，断裂 ​

来源：拉丁语 rumpere，意思是“打破”。rupture 最初指的是身体器官的破裂。

历史典故：disrupt 和 corrupt 都含有“打破”或“破坏”的含义。

常见单词：erupt（喷发）、disrupt（扰乱）、corrupt（腐败）、rupture（破裂）

记忆提示：rupt = 破裂；erupt 是“喷发”或“突然爆发”。

🔹 85. therm – 热 ​

来源：希腊语 thermos，“热的”。用于描述热能和体温的概念。

历史典故：thermometer 是“测量温度的工具”，thermostat 是“温控器”。

常见单词：thermometer（温度计）、thermal（热的）、thermostat（恒温器）

记忆提示：therm = 热；thermometer 是“测量热度的工具”。

🔹 86. cap / cept / ceive – 拿，抓住 ​

来源：拉丁语 capere，意思是“抓住、接收”。

历史典故：capture 原意为“抓住战利品”，accept 和 receive 是“接受”的意思。

常见单词：capture（抓住）、accept（接受）、receive（接收）、concept（概念）

记忆提示：cap = 抓住；capture 是“抓住”的动作或结果。

🔹 87. scrib / script – 写 ​

来源：拉丁语 scribere，“写下”。script 原指古代剧作家的“文稿”。

历史典故：script 是“书面文字的创作”，manuscript 是“手写的书面作品”。

常见单词：describe（描述）、script（剧本）、subscribe（订阅）、inscription（铭文）

记忆提示：scrib = 写；manuscript 是“手写的文稿”。

🔹 88. pel / puls – 推，驱动 ​

来源：拉丁语 pellere，“驱赶、推动”。pulse 是心脏“推动”血液的体现。

历史典故：compel 是“强迫（推动做某事）”，impulse 是“冲动（推动你去做某事）”。

常见单词：compel（强迫）、repel（击退）、expel（驱逐）、impulse（冲动）

记忆提示：pel = 推动；compel 是“迫使（你去做）”。

🔹 89. vers / vert – 转，转变 ​

来源：拉丁语 vertere，“转动”。reverse 是“向后转”。

历史典故：convert 是“完全改变方向或信仰”，advertise 是“朝着目标方向宣传”。

常见单词：convert（转变）、reverse（倒转）、advertise（宣传）、invert（颠倒）

记忆提示：vers = 转；convert 是“完全转变”。

🔹 90. man / manu – 手 ​

来源：拉丁语 manus，“手”。manual 原指用手操作的工具或书。

历史典故：manipulate 是“用手操作”或“巧妙控制”，manufacture 是“通过手工制作”。

常见单词：manual（手册）、manufacture（制造）、manipulate（操纵）、manuscript（手稿）

记忆提示：man = 手；manual 是“用手做的事情”。

🔹 91. spect – 看 ​

来源：拉丁语 spectare，意思是“看，观察”。

历史典故：inspect 是“查看”，spectacle 是“可供观看的表演或景象”。

常见单词：inspect（检查）、spectacle（景象）、spectator（观众）、respect（尊重）

记忆提示：spect = 看；spectacle 是“眼前的景象”。

🔹 92. port – 携带，运输 ​

来源：拉丁语 portare，意为“带，搬运”。

历史典故：export 是“外运”，import 是“进口”，passport 是“护照（带着进入的证件）”。

常见单词：transport（运输）、export（出口）、import（进口）、portable（便携的）

记忆提示：port = 携带；transport 是“携带到别处”。

🔹 93. dict – 说，命令 ​

来源：拉丁语 dicere，“说”。

历史典故：dictate 是“口述命令”，predict 是“预言（说出未来的事情）”。

常见单词：dictionary（字典）、dictate（口述）、predict（预言）、verdict（裁决）

记忆提示：dict = 说；dictionary 是“书面言语的集合”。

🔹 94. aud – 听 ​

来源：拉丁语 audire，意思是“听”。

历史典故：audible 是“能听见的”，auditory 是“听觉的”。

常见单词：audible（可听的）、auditory（听觉的）、audience（观众）、audio（音频）

记忆提示：aud = 听；audible 是“能够听见的”。

🔹 95. cede / ceed / cess – 走，去 ​

来源：拉丁语 cedere，意思是“去，走”。

历史典故：exceed 是“超过”，proceed 是“继续前行”，cessation 是“停止”。

常见单词：exceed（超过）、proceed（继续）、recede（后退）、success（成功）

记忆提示：cede = 去，走；proceed 是“向前走”。

🔹 96. gen – 产生，出生 ​

来源：希腊语 gennan，“产生，生育”。

历史典故：generate 是“生成”，generation 是“代，世代”。

常见单词：generate（生成）、generation（代，世代）、genesis（起源）、gender（性别）

记忆提示：gen = 生成；generation 是“生产出一代”。

🔹 97. inter – 在…之间 ​

来源：拉丁语 inter，“在…之间”。

历史典故：interaction 是“相互作用”，intermediate 是“中间的”。

常见单词：international（国际的）、interact（互动）、intermediate（中间的）、intercept（拦截）

记忆提示：inter = 在…之间；international 是“在不同国家之间”。

🔹 98. biblio – 书 ​

来源：希腊语 biblion，“书”。

历史典故：bibliography 是“书目”，bibliophile 是“爱书的人”。

常见单词：bibliography（书目）、bibliophile（爱书者）、bibliology（书学）

记忆提示：biblio = 书；bibliography 是“书籍的记录”。

🔹 99. auto – 自己 ​

来源：希腊语 autos，“自己”。

历史典故：autograph 是“亲笔签名”，autonomy 是“自我管理”。

常见单词：automatic（自动的）、autograph（亲笔签名）、autonomy（自治）、autocrat（独裁者）

记忆提示：auto = 自己；automatic 是“自己完成的”。

🔹 100. clud / clus – 关闭 ​

来源：拉丁语 claudere，意思是“关闭”。

历史典故：exclude 是“排除”，reclude 是“关闭起来”。

常见单词：exclude（排除）、conclude（结束）、enclose（围住）、include（包括）

记忆提示：clud = 关闭；exclude 是“关闭在外面”。
